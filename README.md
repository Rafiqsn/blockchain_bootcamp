# Blockchain Bootcamp – Frontend (Quiz-Chain)

Frontend React + TypeScript untuk aplikasi **Blockchain Bootcamp / Quiz-Chain Certificate Registry**.  
Aplikasi ini dipakai untuk:
- Register & login user
- Lihat daftar course & chapter
- Kerjakan quiz per chapter
- Claim & lihat certificate on-chain

---

## 1. Prasyarat

Sebelum mulai, pastikan sudah install:

- **Git**
- **Node.js** (minimal versi 18 / 20)
- **npm** (otomatis ikut kalau install Node.js)
- Backend API Blockchain Bootcamp sudah jalan  
  (misal di: `http://127.0.0.1:8000/api/v1` – sesuaikan dengan dokumentasi backend)
